# btn-user-platform

build the project:
```bash
npm run build
```

start the project:
```bash
npm start
```

example settings for VS Code(`.vscode/settings.json`): 
```json
{
    "editor.codeActionsOnSave": {
        "source.fixAll.eslint": true
    },
    "eslint.validate": [
        "javascript"
    ],
    "cSpell.words": [
        "browserslist",
        "paren",
        "parens"
    ]
}
```
